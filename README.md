# 🚀 AI Job Market Intelligence Platform

An end-to-end AI-powered analytics platform that collects real-time job market data, extracts in-demand technical skills using NLP, and intelligently matches resumes with relevant opportunities.

Built to simulate a modern data product workflow — from API ingestion and analytics engineering to AI-driven recommendation systems and interactive dashboards.

---

# 🌟 Key Highlights

- Real-time job market data ingestion using public APIs
- NLP-based skill extraction and market trend analysis
- AI-powered resume matching engine
- Skill gap analysis against current market demand
- Interactive analytics dashboard with filtering and search
- PDF/DOCX/TXT resume parsing pipeline
- Modern multi-page Streamlit application
- Production-style project architecture and modular pipeline

---

# 🧠 Problem Statement

The modern job market changes rapidly, making it difficult for candidates to understand:

- Which skills are currently in demand
- How well their resume aligns with market requirements
- What gaps exist between their profile and target roles

This platform addresses that problem by combining:
- real-time job analytics,
- NLP-based skill intelligence,
- and AI-driven resume matching.

# ⚡ Features

## 📊 Market Intelligence Dashboard

- Hiring trend visualization
- Company hiring analytics
- Geographic hiring distribution

## 🧠 AI Resume Matcher

Upload resumes in:
- PDF
- DOCX
- TXT

The platform:
- parses resume content,
- extracts matching skills,
- compares against live job postings,
- and ranks the best matching opportunities.

---

## 📈 Skill Gap Analysis

Automatically identifies:
- strengths already present in the resume,
- missing high-demand skills,
- opportunities for upskilling based on market trends.
